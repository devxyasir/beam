09

Usage rules

What to never do with the Beam mark and brand.

✕ NEVER

Don't use arbitrary colors on the mark. Only use the brand gradient or mono white/dark.

✕ NEVER

Don't rotate the mark. The beam must always point right — it has direction.

✕ NEVER

Don't stretch or squash. Always scale proportionally.

✕ NEVER

Don't use the gradient mark on white. Use the dark mono version on light backgrounds.

✕ NEVER
beam

Don't use a serif font for the wordmark. Syne only.

✕ NEVER
BEAM

Don't set the wordmark in all-caps with wide tracking. Always lowercase, tight tracking.

Sidebar width300–450px · CSS inherits --vscode-sideBar-background
Glassmorphism Sidebar Rule
Use slight background transparency on the chat panel so the editor underneath bleeds through — creating a sense of layered workspace. Set backdrop-filter: blur(12px) and background: rgba(13,13,15,0.88) on the sidebar root. The code stays visible behind, reinforcing that the chat is a layer over the work, not a separate tool.
